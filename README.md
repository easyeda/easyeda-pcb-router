# easyeda-pcb-router
EasyEDA PCB Router is a CLI client for the PCB auto router which is based on the Freerouting()

NOTE：This project remove many GUI codes, maybe you need to take some time to read the codes.

There is currently no manpower available to complete the documentation for code construction and API integration with EasyEDA. We welcome contributors to submit merge requests.

# Usage
Please refer to the running version and usage instructions: [https://docs.easyeda.com/en/PCB/Route/#local-auto-router](https://docs.easyeda.com/en/PCB/Route/#local-auto-router)
